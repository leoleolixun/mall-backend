## 今日完成

### 订单查询

实现：

```http
GET /api/v1/orders
GET /api/v1/orders/:id
```

要求：

- 只能查询当前用户订单
- 订单详情包含订单明细
- 列表支持分页

### 取消订单

实现：

```http
POST /api/v1/orders/:id/cancel
```

要求：

- 只有待支付订单可以取消
- 取消订单恢复库存
- 状态更新和库存恢复放在同一个事务中

### 模拟支付

实现：

```http
POST /api/v1/orders/:id/pay
```

要求：

- 只有待支付订单可以支付
- 支付后状态变为已支付
- 写入 `paid_at`
