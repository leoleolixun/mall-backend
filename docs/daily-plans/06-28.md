## 今日完成

```text

categories
products
product_skus
```

### 初始化数据

- 创建默认商户：

```text
merchant_id = 1
```

- 初始化几条分类
- 初始化几条商品
- 每个商品至少有一个 SKU

### 商品接口

实现：

```http
GET /api/v1/categories
GET /api/v1/products
GET /api/v1/products/:id
GET /api/v1/products/:id/skus
```

### 分层实现

每个模块至少包含：

```text

repository
model
dto
```

### 商品缓存

- 商品详情优先查 Redis
- Redis 未命中再查 MySQL
- 查到后写入 Redis
- 商品不存在时写入短 TTL 空缓存
