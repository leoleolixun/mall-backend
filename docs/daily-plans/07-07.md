## 第 5 步：实现订单预览 Service

创建：

```text
internal/service/order_service.go
```

接口先定义：

```go
type OrderService interface {
	Preview(ctx context.Context, userID int64, req dto.OrderPreviewRequest) (*dto.OrderPreviewResponse, error)
}
```

预览流程：

```text
1. 校验 address_id
2. 校验 items 非空
3. 查询地址，必须属于当前用户
4. 批量查询 SKU
5. 批量查询 Product
6. 校验 SKU 存在、启用、库存足够
7. 校验商品存在、上架
8. 计算 goods_amount
9. 计算 freight_amount，第一版为 0
10. 计算 discount_amount，第一版为 0
11. 计算 payable_amount
12. 生成 idempotency_token
13. 写入 Redis，设置 TTL
14. 返回预览结果
```

Redis key 建议：

```text
mall:order:idempotency:{user_id}:{token}
```

TTL 建议：

```text
15 分钟
```
