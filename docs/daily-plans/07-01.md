## 今日完成

### 数据表

定义并迁移：

```text
orders
order_items
```

### 订单预览

实现：

```http
POST /api/v1/orders/preview
```

预览逻辑：

- 校验地址属于当前用户
- 校验购物车或请求中的 SKU
- 查询商品和 SKU 当前状态
- 校验库存
- 计算总金额
- 返回商品明细
- 返回地址信息
- 返回幂等 token
- 幂等 token 写入 Redis

### 创建订单

实现：

```http
POST /api/v1/orders
```

创建逻辑必须在事务中完成：

- 校验幂等 token
- 校验地址
- 查询 SKU
- 校验商品状态
- 校验库存
- 创建订单主表
- 创建订单明细
- 保存商品快照
- 保存地址快照
- 扣减库存
- 清理购物车中已下单商品
- 将幂等 token 标记为已使用或绑定订单号
