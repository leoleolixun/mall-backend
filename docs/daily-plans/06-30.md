## 详细执行步骤

### 第 1 步：实现地址模型

创建：

```text
internal/model/address.go
```

字段至少包含：

```text
user_id
receiver_name
receiver_phone
province
city
district
detail
is_default
```

地址所有查询都必须按当前登录用户过滤。

### 第 2 步：实现地址分层

创建：

```text
internal/repository/address_repository.go
internal/service/address_service.go
internal/handler/address_handler.go
```

Service 中处理：

```text
新增地址
修改地址
删除地址
设置默认地址
```

设置默认地址时要注意：

```text
同一个用户只能有一个默认地址
```

建议放在事务里：

```text
先取消当前用户所有默认地址
再把目标地址设为默认
```

### 第 3 步：实现购物车 DTO

购物车请求：

```text
sku_id
quantity
```

购物车响应：

```text
product_id
sku_id
product_name
sku_name
sku_image
price
quantity
subtotal
stock
available
```

### 第 4 步：实现 CartService

购物车不需要 MySQL 表，使用 Redis Hash：

```text
key: mall:cart:{user_id}
field: sku_id
value: quantity
```

Service 需要注入：

```text
Redis client
ProductRepository
```

列表接口流程：

```text
HGETALL mall:cart:{user_id}
取出 sku_id 和 quantity
批量查询 SKU 和 Product
组合返回 DTO
```

### 第 5 步：实现 CartHandler

接口：

```http
GET    /api/v1/cart
POST   /api/v1/cart/items
PUT    /api/v1/cart/items/:sku_id
DELETE /api/v1/cart/items/:sku_id
DELETE /api/v1/cart
```

所有购物车接口都需要 JWT。

### 第 6 步：更新路由

地址和购物车都放在鉴权路由组里：

```text
auth := api.Group("")
auth.Use(middleware.Auth())
```

然后注册：

```text
/addresses
/cart
```

### 第 7 步：验收

先登录拿 token，再验证：

```bash
curl -H "Authorization: Bearer <token>" http://127.0.0.1:8080/api/v1/addresses
curl -H "Authorization: Bearer <token>" http://127.0.0.1:8080/api/v1/cart
redis-cli -n 5 hgetall mall:cart:<user_id>
```

### 购物车接口

实现：

```http
GET    /api/v1/cart
POST   /api/v1/cart/items

DELETE /api/v1/cart/items/:sku_id

```

- 商品 ID
- SKU ID
- 商品名称
- SKU 名称
- SKU 图片

- 数量
- 小计
- 商品状态
- 库存状态
