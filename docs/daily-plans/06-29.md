## 详细执行步骤

### 第 1 步：定义用户模型

创建：

```text
internal/model/user.go
internal/model/user_auth.go
```

`users` 保存用户主体信息，`user_auths` 保存登录方式。

今天不要拆成 `password_users`、`wechat_users` 两套用户系统。正确方向是：

```text
users
user_auths
```

两种登录方式最终都映射到同一个 `users.id`。

### 第 6 步：实现 AuthService

创建：

```text
internal/service/auth_service.go
```

实现：

```text
注册
账号密码登录
微信小程序登录
刷新 token
退出登录
获取当前用户
```

账号密码注册流程：

```text
校验 username/password
查询 username 是否已存在
bcrypt hash 密码
创建 users
创建 user_auths(provider=password)
```

微信小程序登录流程：

```text
接收 openid
查询 user_auths(provider=wechat_mini_program, provider_uid=open_id)
存在则返回对应用户
不存在则创建 users + user_auths
签发 JWT
```

### 第 7 步：实现 JWT 中间件

创建：

```text
internal/middleware/auth.go
```

中间件负责：

```text
读取 Authorization: Bearer <token>
解析 token
把 user_id 写入 gin.Context
解析失败时返回未登录
```
